Core Features

Everything in one platform

Built for both listeners and creators — with a full audio marketplace, real payment flow, and a powerful admin panel.

Professional Audio Player
Full-featured audio player with instant streaming, smart buffering, crossfade, shuffle & repeat, playback speed control, and Media Session API for lock-screen controls on every device. A 5-band equalizer with real-time frequency spectrum, reverb, stereo balance, and 12 presets is built right in.

Upload Audio — Any User
Any registered user can upload MP3, WAV, OGG, M4A, AAC, FLAC (up to 100 MB) with a cover thumbnail. Set visibility as Free, Paid, or Private. Files go directly to Cloudinary with auto folder organization.

Authentication

Secure auth, done right

Multiple login methods, timing-safe checks, OTP reset, and Google OAuth — all with httpOnly JWT cookies.

Email / Password + Google OAuth
Register with email/password or sign in with Google (full OAuth 2.0 flow). Auto-detects Gravatar avatars on registration. Timing-safe login prevents user enumeration attacks.
Google OAuth 2.0
JWT — 7-Day Sessions
JWT tokens are delivered as httpOnly cookies (CSRF-safe) and as JSON body. Tokens expire after 7 days. Optional auth middleware serves public endpoints while still enriching context for logged-in users.
httpOnly Cookie
OTP Password Reset
6-digit OTP via email (30-minute expiry). Privacy-safe response prevents user enumeration. Expired OTPs auto-cleaned hourly. Full 3-step flow: request → verify → reset.
Resend API
Profile & Avatar Management
Users can update name, change password, upload a custom avatar (auto face-crop to 200×200 via Cloudinary) or clear it. Old Cloudinary assets auto-deleted on replace.
Auto Face-Crop
Country Auto-Detection
Cloudflare CF-IPCountry header (primary) → ip-api.com fallback → DEFAULT. Country code stored per user session and used to show relevant payment methods in the purchase modal.
Smart UX
Rate Limiting — Tiered
General API: 200 req/15 min. Login: 20 req/15 min. Register: 10 req/15 min. OTP: 5 req/hour. Reset: 10 req/15 min. Per-endpoint limits prevent brute force and abuse.
DDoS Protection

Security

Secure by default

Security is not an afterthought — it is baked into every layer of the stack.

Helmet — Full CSP
Content Security Policy with strict allowlists for scripts, styles, fonts, images, and media. frameAncestors 'none', objectSrc 'none'. Manual Permissions-Policy header for camera, mic, geolocation.
XSS Protection
httpOnly JWT Cookies
Tokens stored in httpOnly, Secure, SameSite=None cookies — not accessible via JavaScript. Both cookie and Bearer header supported for flexibility.
CSRF Safe
Tiered Rate Limiting
Different rate limits per endpoint. Auth endpoints are stricter. Uses standardHeaders (RateLimit-*) for transparent client communication.
Brute Force Guard
Input Validation & Sanitization
All inputs trimmed, sliced to max length, and type-checked before DB queries. Parameterized queries everywhere — zero raw string interpolation. Email, password, and name format validated on every route.
SQL Injection Proof
Per-Track Access Control
Public, paid, and private visibility enforced server-side on every audio endpoint. Paid audio_url is NULL in all list responses until purchase is completed. Private tracks blocked for everyone except owner and admin.
Zero URL Leaks
bcrypt + Timing-Safe Login
Passwords hashed with bcrypt (cost 10). Login always runs bcrypt even for non-existent users — prevents timing-based user enumeration. OTP response always identical regardless of email existence.
Anti-Enumeration
